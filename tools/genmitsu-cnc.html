What You Need to Learn

  • Use approved workholding.
  • Set zero accurately.
  • Check bit and material compatibility.
  • Monitor sound, chips, and movement.

Certification Focus

This page prepares you for the hands-on check. Passing the quiz does not replace the instructor sign-off.

Ask before acting: If the setup does not match the training, stop and ask your instructor.
Real FabLab Scenario: The material shifts slightly during a cut. What should happen? Stop immediately and improve workholding.

Student Procedure

  1. Review the toolpath and stock setup.
  2. Secure the material.
  3. Install the approved bit.
  4. Set the work zero.
  5. Verify clamps are outside the toolpath.
  6. Start under supervision and monitor the first pass.
  7. Stop for chatter, burning, or unexpected motion.

Before You Start

  • Toolpath previewed.
  • Clamps clear.
  • Zero set.
  • Machine monitored.

Stop Immediately If...

  • The tool makes an unusual sound.
  • The material moves unexpectedly.
  • You smell burning, see smoke, or notice heat where it should not be.
  • You are unsure what the next step should be.
