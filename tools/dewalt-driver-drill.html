What You Need to Learn

  • Match bit to fastener or drill size.
  • Secure material.
  • Use controlled pressure.
  • Avoid over-torquing fasteners.

Certification Focus

This page prepares you for the hands-on check. Passing the quiz does not replace the instructor sign-off.

Ask before acting: If the setup does not match the training, stop and ask your instructor.
Real FabLab Scenario: A screw starts stripping. What should happen? Stop and check bit size, pressure, alignment, clutch, and pilot hole.

Student Procedure

  1. Select the correct bit and seat it fully.
  2. Secure the workpiece.
  3. Set drill/driver mode and clutch if needed.
  4. Keep the bit aligned with the work.
  5. Use steady pressure and stop before stripping the fastener.
  6. Return the bit and tool to storage.

Before You Start

  • Correct bit.
  • Material secured.
  • Hands clear behind work.
  • Battery/tool condition checked.

Stop Immediately If...

  • The tool makes an unusual sound.
  • The material moves unexpectedly.
  • You smell burning, see smoke, or notice heat where it should not be.
  • You are unsure what the next step should be.
