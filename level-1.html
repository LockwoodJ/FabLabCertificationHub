Level 1 • Core Access

Start here: safety, awareness, and basic tool habits.

Level 1 is required before you complete any FabLab work. Your goal is to show that you can enter the space, recognize hazards, use basic tools, and clean up responsibly.

Earn This Level By Showing You Can...

  • Find emergency equipment.
  • Set up a clean, safe workspace.
  • Use hand tools with control.
  • Reset the workspace for the next group.
