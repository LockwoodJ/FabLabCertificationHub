What You Need to Learn

  • Select the correct bit.
  • Clamp the workpiece.
  • Set the table and depth safely.
  • Keep hands away from the rotating bit.

Certification Focus

This page prepares you for the hands-on check. Passing the quiz does not replace the instructor sign-off.

Ask before acting: If the setup does not match the training, stop and ask your instructor.
Real FabLab Scenario: A small part begins spinning on the drill press table. What should happen? Stop immediately; never grab it by hand.

Student Procedure

  1. Choose the correct drill bit and confirm it is tight in the chuck.
  2. Remove the chuck key if applicable.
  3. Clamp the workpiece to the table.
  4. Set table height and depth stop if needed.
  5. Turn on the drill and feed slowly with control.
  6. Wait for the bit to stop before removing the part.

Before You Start

  • Work clamped.
  • Correct bit installed.
  • Hair/clothing secured.
  • Hands clear.

Stop Immediately If...

  • The tool makes an unusual sound.
  • The material moves unexpectedly.
  • You smell burning, see smoke, or notice heat where it should not be.
  • You are unsure what the next step should be.
