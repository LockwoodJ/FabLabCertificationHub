What You Need to Learn

  • Export a clean STL or 3MF from CAD.
  • Choose orientation, layer height, infill, and supports intentionally.
  • Preview the sliced toolpath before printing.
  • Watch the first layer and stop failed prints early.
  • Remove prints safely and reset the machine area.

Real FabLab Scenario: A classmate starts a long print without watching the first layer. What should you do? Pause the process, check the first layer expectation, and involve the instructor if the print is already failing.

Student Procedure

  1. Confirm the design is school appropriate, printable, and meets assignment constraints.
  2. Open the model in Bambu Studio or the approved slicer profile.
  3. Choose the correct material, printer profile, layer height, infill, supports, and bed adhesion settings.
  4. Preview the sliced file and check for floating parts, excessive supports, or unreasonable print time.
  5. Start or submit the print through the approved classroom process.
  6. Stay for the first layer and stop the job if adhesion, scraping, spaghetti, or unusual noise occurs.
  7. Remove the part only when safe, clean the plate/area, and document the print result.

Before You Start

  • Nozzle and bed are not touched while hot.
  • Build plate is clean and properly seated.
  • Filament type matches the slicer profile.
  • First layer is smooth, attached, and not scraping.

Stop Immediately If...

  • The tool makes an unusual sound.
  • The material moves unexpectedly.
  • You smell burning, see smoke, or notice heat where it should not be.
  • You are unsure what the next step should be.
