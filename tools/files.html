What You Need to Learn

  • Secure material before filing.
  • Use a file handle.
  • File with controlled strokes.
  • Clean metal/plastic dust safely.

Certification Focus

This page prepares you for the hands-on check. Passing the quiz does not replace the instructor sign-off.

Ask before acting: If the setup does not match the training, stop and ask your instructor.
Real FabLab Scenario: The workpiece slides while filing. What should happen? Stop and clamp or secure the workpiece.

Student Procedure

  1. Choose the correct file shape and coarseness.
  2. Secure the workpiece.
  3. Hold the handle and file with controlled forward strokes.
  4. Keep fingers away from sharp edges.
  5. Check the edge frequently rather than over-filing.
  6. Brush debris away using the approved method.

Before You Start

  • Work secured.
  • Handle used.
  • Controlled strokes.
  • Debris cleaned.

Stop Immediately If...

  • The tool makes an unusual sound.
  • The material moves unexpectedly.
  • You smell burning, see smoke, or notice heat where it should not be.
  • You are unsure what the next step should be.
